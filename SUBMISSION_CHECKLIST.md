## COMMON ISSUES & SOLUTIONS

### Build Fails
```
Solution: Clear cache and rebuild
rm -rf node_modules
npm install
npm run build
```

### App Rejected for Privacy Policy
```
Solution: Add privacy policy link to app store listing
Use PRIVACY_POLICY.md provided
```

### App Rejected for Permissions
```
Solution: Explain why app needs camera access
Add permission explanation to app store listing
```

### Payment Setup Issues
```
Solution: Verify bank account details
Ensure account is in your name
Check with your bank for any blocks
```

### Screenshots Rejected
```
Solution: Make sure screenshots:
- Are correct dimensions
- Show actual app features
- Include app name or branding
- Are high quality (no blurry images)
```
